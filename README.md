# Builded-a-safe-and-isolated-cybersecurity-practice-environment

## Objective
Build an isolated penetration testing lab using VMware, Kali Linux, and OWASP Juice Shop.

## Architecture
- VMware Workstation
- Kali Linux (Attacker)
- OWASP Juice Shop (Docker)
- NAT + Host-Only Network

## Tools Used
- Nmap
- Burp Suite
- Wireshark
- Docker

## Outcome
Successfully deployed vulnerable web app and performed reconnaissance and traffic interception.
